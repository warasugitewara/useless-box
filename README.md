# useless-box

Arduino UNO 3 で実装した「意味のない箱 (Useless Box)」です。  
スイッチを押すと箱のフタが開いて腕が出てきてスイッチを押し戻し、またフタが閉まります。

> 課題研究作品

---

## ハードウェア構成

| パーツ | 型番 | 役割 | ピン |
|--------|------|------|------|
| マイコン | Arduino UNO 3 | 制御 | — |
| サーボ（フタ） | SG90 | フタの開閉 | D5 |
| サーボ（腕） | MG996R | スイッチを押す腕 | D6 |
| スイッチ | — | トリガー入力 | D12 |
| 電源 | 単三電池 × 3 | 全体給電 | — |
| コンデンサ | 470µF × 2 (並列) | MG996R 起動スパイク吸収 | MG996R 根本に並列 |

### 配線

```
電池 R+ ── 主スイッチ ──→ Arduino 5V / サーボ VCC
電池 B-  ──────────────→ Arduino GND / サーボ GND

Arduino D5  →  SG90  オレンジ（信号）
Arduino D6  →  MG996R 黄（信号）
Arduino D12 →  スイッチ 白
スイッチ 黒  →  GND

コンデンサ 470µF × 2（並列）
  + 側 → MG996R 赤（VCC）
  - 側 → MG996R 茶（GND）
```

---

## バリエーション（5種ランダム）

| Case | 名前 | 内容 |
|------|------|------|
| 1 | ノーマル | 迷わずスパッと処理 |
| 2 | ためらいがち | フタが途中で止まり、引き戻してから諦めて開ける |
| 3 | 即ブチ切れ | 全力最速で反応 |
| 4 | フェイント2連 | 2回チラ見せしてから本番動作 |
| 5 | 連続ノック | 腕を2回叩きつける |

電源投入のたびにランダムシードが変わるため、毎回異なる順序でバリエーションが出ます。

## 角度設定

```cpp
const int lidClosed = 90;   // フタが閉まる角度
const int lidOpen   = 3;    // フタが開く角度
const int armRetracted = 3;    // 腕が格納された角度
const int armExtended  = 110;  // 腕が伸びた角度
```

サーボの取り付け方向によって `lidClosed` / `lidOpen` の値が変わります。

---

## 注意事項

- `A0` ピンは **何も接続しないでください**（ランダムシードに使用）
- MG996R は起動時に最大 2.5A の電流を消費します。電源容量に注意してください
- 電池で運用する場合、アルカリ単三では電圧降下が起きやすいです。コンデンサを MG996R 根本に並列接続することで軽減できます
