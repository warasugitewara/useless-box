# useless-box

スイッチを押すと箱のフタが開き、腕が出てきてスイッチを押し戻す——それだけの箱。  
Arduino UNO 3 で実装した課題研究作品です。

---

## 回路図

<img src="docs/circuit.svg" width="720" alt="回路図">

### 構成部品

| パーツ | 型番 | 役割 | ピン |
|--------|------|------|------|
| マイコン | Arduino UNO 3 | 制御 | — |
| サーボ (フタ) | SG90 | フタの開閉 | D5 |
| サーボ (腕) | MG996R | スイッチを押す腕 | D6 |
| スイッチ | — | トリガー入力 | D12 |
| 電源 | 単三電池 × 3 | 全体給電 (4.5V) | — |
| コンデンサ | 470µF × 2 (並列) | MG996R 起動スパイク吸収 | MG996R 根本に並列 |

### 配線メモ

- 電源系はすべてはんだ付け＋ホットボンド固定（ブレッドボードなし）
- コンデンサは MG996R の VCC(赤)・GND(茶) 根本に直接並列接続
- `A0` ピンは **未接続のまま**（ランダムシード用に浮遊ノイズを利用）

## バリエーション

<img src="docs/variations.svg" width="680" alt="バリエーション一覧">

| Case | 名前 | 内容 |
|------|------|------|
| 1 | ノーマル | 迷わずスパッと処理する |
| 2 | ためらいがち | フタが途中で止まり、引き戻してから諦めて開ける |
| 3 | 即ブチ切れ | 全力最速で反応 |
| 4 | フェイント2連 | 2回チラ見せしてから本番動作 |
| 5 | 連続ノック | 腕を2回叩きつける |

電源投入のたびに `randomSeed(analogRead(A0))` でシードが変わり、  
毎回異なる順序でバリエーションが出ます。

## コード

```cpp
// 角度設定（SG90 の取り付け方向に合わせて調整済み）
const int lidClosed    = 90;   // フタが閉まる角度
const int lidOpen      = 3;    // フタが開く角度
const int armRetracted = 3;    // 腕が格納された角度
const int armExtended  = 110;  // 腕が伸びた角度

// 動作完了待機時間
const int LID_MOVE_MS = 200;   // SG90:   90° 動ききる余裕時間
const int ARM_MOVE_MS = 420;   // MG996R: 110° 動ききる余裕時間
```

`sweep()` 関数で SG90 を 1°ずつ動かすことで、ためらいやフェイントを表現しています。  
MG996R はトルクが強く途中停止が不安定なため `write()` 直接制御を使っています。

---

## 注意事項

- MG996R は起動時に最大 2.5A の電流を消費します。電源容量に注意してください
- 単三電池 3本 (4.5V) はサーボ仕様下限 (4.8V) をわずかに下回るため、電圧降下対策としてコンデンサを並列接続しています
- `A0` ピンには何も接続しないでください
