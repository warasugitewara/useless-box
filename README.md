# useless-box

スイッチを押すと箱のフタが開き、腕が出てきてスイッチを押し戻す——それだけの箱。  
Arduino UNO 3 で実装した課題研究作品です。

### 構成部品

| パーツ | 型番 | 電源系統 | 役割 | ピン |
|--------|------|----------|------|------|
| マイコン | Arduino UNO 3 | 電源A (4.5V) | 制御 | — |
| サーボ (フタ) | SG90 | 電源A (4.5V) | フタの開閉 | D5 |
| サーボ (腕) | MG996R | **電源B (6.0V)** | スイッチを押す腕 | D6 |
| スイッチ | — | — | トリガー入力 | D12 |
| 電源A | 単三電池 × 3本 | — | Arduino・SG90 給電 | — |
| 電源B | 単三電池 × 4本 | — | **MG996R 専用給電** | — |
| コンデンサ | 470µF × 2 (並列) | 電源B (6V系) | MG996R 起動スパイク吸収 | MG996R 根本に並列 |

### 2電源構成について

MG996R 動作時（最大 2.5A）の電圧降下で回路全体がリセットされる問題を解決するため、電源を分離しています。

| 系統 | 電池 | 電圧 | 対象 |
|------|------|------|------|
| 電源A | 単三 × 3本 | 4.5V | Arduino UNO 3、SG90 |
| 電源B | 単三 × 4本 | **6.0V** | MG996R のみ |

> **⚠ GND は必ず両系統で共通接続すること**  
> Arduino D6 → MG996R 信号線の基準電位を合わせるために必須。GND を共通にしないと MG996R が正常動作しない。

### 配線メモ

- 電源系はすべてはんだ付け＋ホットボンド固定
- コンデンサは MG996R の VCC(赤)・GND(茶) 根本に直接並列接続（6V 系統）
- `A0` ピンは **未接続のまま**（ランダムシード用に浮遊ノイズを利用）

<img src="docs/variations.svg" width="680" alt="バリエーション一覧">

| Case | 名前 | 内容 |
|------|------|------|
| 1 | ノーマル | 迷わずスパッと処理する |
| 2 | ためらいがち | フタが途中で止まり、引き戻してから諦めて開ける |
| 3 | 即ブチ切れ | 全力最速で反応 |
| 4 | フェイント2連 | 2回チラ見せしてから本番動作 |
| 5 | 連続ノック | 腕を2回叩きつける |

電源投入のたびに `randomSeed(analogRead(A0))` でシードが変わり、毎回異なる順序でバリエーションが出ます。

## コード

```cpp
// 角度設定（SG90 の取り付け方向に合わせて調整済み）
const int lidClosed    = 90;   // フタが閉まる角度
const int lidOpen      = 3;    // フタが開く角度
const int armRetracted = 3;    // 腕が格納された角度
const int armExtended  = 110;  // 腕が伸びた角度

// 動作完了待機時間
const int LID_MOVE_MS = 200;   // SG90:   90° 動ききる余裕時間（4.5V基準）
const int ARM_MOVE_MS = 420;   // MG996R: 110° 動ききる余裕時間（6V では 300ms 前後まで短縮可）
```

`sweep()` 関数で SG90 を 1°ずつ動かすことで、ためらいやフェイントを表現しています。  
MG996R はトルクが強く途中停止が不安定なため `write()` 直接制御を使っています。

---

## 注意事項

- MG996R の動作電圧は **4.8〜7.2V**。電源Bの6.0Vはこの範囲内
- MG996R は起動時に最大 2.5A の電流を消費します。電源Aと分離することで他の回路への影響を排除
- `ARM_MOVE_MS = 420` は 4.8V 基準の余裕値。6V 給電では MG996R が速くなるため実機で短縮を確認してください
- `A0` ピンには何も接続しないでください
